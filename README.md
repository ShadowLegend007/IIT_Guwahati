# Incognito - AI-Native Ingredient Co-Pilot

Incognito is an intelligent, AI-driven application designed to decode product ingredients instantly. It acts as a co-pilot for consumers, reading labels to provide clear, actionable insights, safety scores, and health recommendations without the cognitive load.

## ‚ú® Features

- **Ingredient Analysis**: Instantly breaks down complex ingredient lists into clear categories.
- **Safety Scoring**: Provides an AI-calculated safety score based on ingredient profiles.
- **Smart Insights**: Highlights key pros, cons, and potential health impacts (e.g., bloating, allergens).
- **Age-Specific Advice**: tailored recommendations for Adults, Teens, Children, and Babies.
- **Visual Narratives**: Beautiful, data-rich visualizations including pie charts and safety meters.
- **Glassmorphism UI**: A premium, modern, and fluid user interface.
